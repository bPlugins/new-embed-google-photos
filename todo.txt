-- Performance & SEO (retention) --
[x] Lazy-load images + blur/low-res placeholder
       -> benefit: page feels instant (blurred preview, then fade-in); off-screen
          images don't download until scrolled to -> lighter page, saves mobile data
       -> benefit: fixed-ratio boxes stop layout jumping (better CLS / Core Web Vitals)
       -> benefit: lower bounce, visitors stay longer (retention)
       -> সুবিধা: পেজ সাথে সাথে ভরাট দেখায় (ঝাপসা প্রিভিউ, পরে fade-in); স্ক্রিনের
          বাইরের ছবি scroll না করা পর্যন্ত নামে না -> পেজ হালকা, মোবাইলে ডেটা বাঁচে
       -> সুবিধা: নির্দিষ্ট-অনুপাতের বক্সে লেআউট লাফায় না (CLS / Core Web Vitals ভালো)
       -> সুবিধা: bounce কমে, ভিজিটর বেশিক্ষণ থাকে (retention)
[x] Responsive srcset (multiple image sizes, not just =w2048)
       -> benefit: browser downloads the size it actually needs (small on phones,
          large on desktop/retina) -> much faster loads, less bandwidth
       -> benefit: faster LCP -> better Core Web Vitals & Google ranking signal
       -> সুবিধা: ব্রাউজার দরকার অনুযায়ী সাইজ নামায় (ফোনে ছোট, ডেস্কটপ/রেটিনায় বড়)
          -> অনেক দ্রুত লোড, কম bandwidth
       -> সুবিধা: দ্রুত LCP -> ভালো Core Web Vitals ও Google র‍্যাঙ্কিং সিগন্যাল
[x] Schema.org ImageObject markup (Google Image SEO)
       -> benefit: images become eligible for Google Images / rich results ->
          extra free search traffic from image search
       -> benefit: captions, titles and licence info understood by search engines
       -> সুবিধা: ছবি Google Images / rich results-এ দেখানোর যোগ্য হয় -> ইমেজ সার্চ
          থেকে বাড়তি ফ্রি ট্রাফিক
       -> সুবিধা: ক্যাপশন, টাইটেল ও লাইসেন্স তথ্য সার্চ ইঞ্জিন বুঝতে পারে
[x] Per-image alt text editing (accessibility + SEO)
       -> benefit: screen readers describe each image (accessibility / WCAG)
       -> benefit: alt text is a direct Google Images ranking factor -> more SEO reach
       -> benefit: shown as fallback text if an image ever fails to load
       -> সুবিধা: স্ক্রিন রিডার প্রতিটি ছবি বর্ণনা করে (accessibility / WCAG)
       -> সুবিধা: alt টেক্সট Google Images-এ সরাসরি র‍্যাঙ্কিং ফ্যাক্টর -> বেশি SEO রিচ
       -> সুবিধা: ছবি লোড না হলে fallback টেক্সট হিসেবে দেখায়
